What Is a Spring Loaded Door Hinge?

A spring loaded door hinge — also called a door closer hinge or automatic closer hinge — combines the function of a standard door hinge with an integrated closing mechanism. When the door is opened and released, the spring or hydraulic system returns it to the closed position automatically, without requiring manual effort.

Construction Specifier magazine identifies five basic types of door closers for commercial applications. However, a sixth category — the automatic door closer hinge (also called a spring hinge with hydraulic control) — is often overlooked. This category combines:

  • Door hinge function — supports door weight and pivot movement
  • Spring closer function — provides the closing force via internal spring
  • Hydraulic damper function — controls closing speed to prevent slamming
  • Optional hold-open function — allows door to stay open at a set angle when needed

Three Primary Reasons to Choose Spring Loaded Door Hinges

1. Fire Safety Compliance

Building codes universally require self-closing devices on fire-rated doors. Spring loaded hinges satisfy this requirement as an NFPA 80-approved self-closing device. Specific fire code requirements that mandate self-closing doors include:

  • Doors in smoke partitions and smoke barriers
  • Egress path doorways in commercial buildings
  • Occupied spaces containing combustible materials
  • Doors between living areas and attached garages in residential buildings

Waterson spring loaded hinges carry a UL-listed 3-hour fire rating — the highest available — making them suitable for any fire-rated assembly from 20 minutes to 3 hours.

2. Hygiene and Pathogen Reduction

Modern building design increasingly prioritizes reducing pathogen transmission through high-touch surfaces. Spring loaded door hinges enable hands-free door operation: the user pushes or pulls the door open, and it closes automatically — eliminating the need to touch the door again to close it.

This is particularly important in:

  • Healthcare facilities — hospitals, clinics, dental offices
  • Food service environments — commercial kitchens, restaurants
  • Educational facilities — schools, universities
  • High-traffic public areas — restrooms, lobbies, corridors

3. Convenience and Accessibility

Manual door closing presents practical challenges for users carrying items, attending to children, or using mobility devices. Spring loaded hinges eliminate the need to manually close doors, improving convenience for all users and enabling accessibility for individuals with limited upper-limb mobility.

ADA and Accessibility Requirements

Spring loaded door hinges used on accessible routes must comply with ADA and ICC A117.1 standards. Waterson hydraulic spring hinges offer field-adjustable closing speed and spring tension to meet these requirements:

  • Maximum opening force: 5 lbf for interior doors
  • Minimum closing time: 5 seconds from 90° to 12°
  • Clear opening width: Minimum 32" when door is at 90°

The hydraulic mechanism allows precise adjustment to satisfy both fire code closing reliability and ADA force/speed limits on the same door.

NFPA 80 Fire Code Compliance

For a fire door assembly to pass NFPA 80 inspection, the self-closing device must meet these requirements:

  • Self-closing: Door must close and latch from any open position
  • UL listing: Hardware must be listed and labeled for the door assembly fire rating
  • Positive latching: Door must latch securely upon closing
  • Annual inspection: NFPA 80 requires annual fire door inspections to verify self-closing function

Frequently Asked Questions

Q: What is a spring loaded door hinge?

A spring loaded door hinge is a combination device that integrates a door hinge with an automatic closing mechanism — either a coil spring or hydraulic spring — so the door returns to the closed position after each opening. Waterson spring loaded hinges use a hydraulic mechanism with adjustable closing speed for smooth, controlled automatic closure.

Q: Are spring loaded door hinges required by fire code?

Yes. NFPA 80 mandates self-closing devices on all fire-rated door assemblies. Building codes such as the NYC Building Code require self-closing fire doors in smoke partitions, egress paths, spaces with combustible materials, and between living areas and attached garages. Spring loaded hinges satisfy this requirement as an approved self-closing device.

Q: Do spring loaded door hinges meet ADA requirements?

Yes, Waterson spring loaded door hinges are ADA and ICC A117.1 compliant. The adjustable hydraulic mechanism allows the opening force to be set below the ADA maximum of 5 lbf for interior doors, and the closing speed can be set to meet the minimum 5-second requirement from 90° to 12° — balancing fire safety with accessibility.

Q: How do spring loaded door hinges improve hygiene?

Spring loaded door hinges enable hands-free door operation by automatically closing the door after each use. This minimizes contact with door hardware surfaces, reducing the transmission of pathogens — a critical consideration in hospitals, food service facilities, schools, and post-pandemic building design.

Q: What is the weight capacity of spring loaded door hinges?

Waterson spring loaded door hinges support doors weighing over 260 lbs when three hinges are used, and up to 440 lbs when four hinges are used. The minimum hinge size is 4"×4". Stainless steel construction ensures long-term durability in high-traffic commercial and residential applications.
